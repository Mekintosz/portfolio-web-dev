Black Olives

Responsive Restaurant Website

View Live Demo View on GitHub

Single-page, mobile-first restaurant site with booking and newsletter.

Project Overview

Black Olives is a modern, fully responsive restaurant website designed with a mobile-first approach. Customers can browse the menu, make reservations, and subscribe to updates through an elegant, performant UI.

Key Features

Mobile-First: Pixel-perfect across phones, tablets, and desktops

Booking Form: Real-time input validation and clear feedback

Smooth Navigation: SPA-style anchors with animated scrolling

Newsletter Signup: Validated form for email capture

Performance: Optimized images and efficient code

Semantic HTML: Better SEO and screen reader support

The Challenge

Create a single-page experience that communicates brand quality while providing essential functionality like reservations and menu browsing - without sacrificing performance or accessibility across devices.

The Solution

Built with semantic HTML5, CSS Grid/Flexbox, and smooth anchor navigation. Implemented custom validation for table booking and newsletter signup. Images are optimized for speed, and layout adapts beautifully from mobile to desktop.

What I Learned

  • Accessible, user-friendly form patterns
  • Image optimization for faster loads
  • CSS Grid/Flexbox layout strategies
